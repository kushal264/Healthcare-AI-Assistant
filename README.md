# 🏥 Healthcare AI Assistant

An offline medical assistant using TF-IDF and cosine similarity to reduce hallucination in healthcare AI.

## 🚀 Features
- No API required (offline)
- TF-IDF based retrieval
- Cosine similarity matching
- Safety layer to prevent harmful responses
- Confidence scoring

## 🧠 Tech Stack
- Python
- Scikit-learn
- NumPy

## ▶️ How to Run
```bash
python main.py
